# AndroidGroceryStore
an android project like a grocery store

## 点赞红心贝塞尔曲线
点击页面添加HeartView，不过是用ImageView实现的

## 显示顶层的Activity
这个小工具很实用，通过悬浮窗显示顶层的Aty，开发中很实用

## 波浪
贝塞尔曲线实现波浪，代码很简单，通俗易懂，可调波浪高度 波浪速度 波浪个数

## QQ消息拖拽
就是个简单的自定义view 类似的一个效果，再封装下，即可使用

## 沉浸式状态栏
适配高低版本的状态栏沉浸式的实现
